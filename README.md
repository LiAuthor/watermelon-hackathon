# 识瓜
    2019年郑州GDG黑客松活动最佳创意奖项目

# 项目功能
    输入图片（包括摄像头现场拍照图片或手动选取图片），自动判别图片中是否含有西瓜。
    
# 技术选型
    web页面 + 后端 + 机器学习
    * 界面： html5 vue
    * 后端： python django django-restful-framework
    * 数据库： 不使用数据库
    * 机器学习： python 不使用AI框架
